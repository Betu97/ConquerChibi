0. Proposal = portada del GDD (maca) + introducció

1. Introducció. Creadors + info de cada un // High concept + pitch = primeres línies que descriuen en què consisteix el joc

2. Món de joc. Ambientació (època, tipus de món (steampunk, infernal...)... // Conjunt d'elements visuals que embolcallen l'acció amb un estil artístic concret) + Personatges principals (nom, biografia, motivacions, ètica i moral, relació amb el món i amb altres personatges, evolució durant el joc, descripció física, imatges o dibuixos conceptuals) + Enemics/NPC (igual que els personatges principals) + Escenes/nivells (no confondre amb Level Dessign // nom dels escenaris, ¿per què el jugador es troba dins aquest escenari?, descripció física de l'escena, tonalitat emocional d'escena, clima, hora del dia, estació de l'any, NPC que hi ha a l'escena, color general, efectes sonors i música)

3. Gameplay (mecàniques (comportaments interns normalment desconeguts per al jugador // físiques, economia interna, mecàniques de progressió, maniobres de tàctica, interacció social) // metagaming&metagameplay (merchandising, YouTube, comunitat, esdeveniments) // gamefeel (a través de la càmera, sons...) // regles (instruccions bàsiques)) -> (acció -> mecànica (reacció) -> conseqüència) // Acabar amb la regla més important del joc (el seu objectiu)

4. Interacció d'usuari / UI (tipus d'UI (diegètiques (integrades en el món del joc, no sobre la càmera), no diegètiques (tota la HUD està enganxada a la càmera), meta interficies (mescla entre les dos, HUD pegada a la càmera, però que interactua amb l'entorn) i interficies espaials (integrada en el joc, però com a elements 3D (p.e., outlines)), wireframes (cóm seràn les pantalles visualment), flowchart (interaccions de menús (draw.io)), button mapping) 

5. Level Design (construcció del disseny del joc en la seva totalitat // investigar, localitzar, fotografiar, captar la llum // duració nivell, objectiu del nivell, inici i final del nivell, punts de spawn, enemics i/o NPCs, per on es pot caminar, objectes per a agafar, zones secretes o objectes amagats (easter eggs), assets per nivell, altres (limitacions, restriccions, comentaris de dissenyador), estructura, estructura (lineal, bifurcada, >nodal< (el nostre), aniuada) // ficar llegenda i línies d'acció en els esquemes (espais interessants?, divertit?, recompensa?, dificultat?, coherent en la història?, just amb el jugador?)

6. Project Plan

7. Prototip (no cal)
